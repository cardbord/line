# line
an esoteric low-level programming language made completely of straight line symbols!

## well... kind of
symbols include ` /, \, <, >, ', ", =, _, -, |, ¬, and +! `
not _all_ of them are straight lines, so bare with me here!

### Hello, world!

Here's a hello world program, written in line.

```
+¬+
_<<_=/\//\///=
_<<_=/\\//\/\=
_<<_=/\\/\\//=
_<<_=/\\/\\//=
_<<_=/\\/\\\\=
_<<_=/\/\\//=
_<<_=/\/////=
_<<_=/\\\/\\\=
_<<_=/\\/\\\\=
_<<_=/\\\//\/=
_<<_=/\\/\\//=
_<<_=/\\//\//=
_<<_=/\////\=
```
